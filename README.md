# Todo List App
Please design a system to manage individual to do lists.

## Features 
1. Login
2. Add a new list
3. Add an item to a list.
4. Edit list details (name for example).
5. Edit list item.
6. Delete list.
7. Delete list item.

## Run the app

Run the app by running 

```bash
$ pnpm dev
```

## Testing the app

Test everything

```bash
$ pnpm test:unit
```

Test the frontend

```bash
$ pnpm --filter frontend test:unit
```

Test the backend

```bash
$ pnpm --filter backend test:unit
```
